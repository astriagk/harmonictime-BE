# E-commerce App (TypeScript)

This project is a basic setup for an E-commerce API built with TypeScript, Node.js, and MongoDB.

## How to Use

1. Clone the repository and navigate to the project directory.
2. Install dependencies: `npm install`.
3. Set up your `.env` file with the correct database URI and other configurations.
4. Run the application in development mode: `npm run dev`.
5. Build the project: `npm run build`.
6. Start the server: `npm start`.
